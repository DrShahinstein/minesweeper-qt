# Minesweeper

Minesweeper game written in C++ using Qt6.

https://github.com/user-attachments/assets/66a7b4fd-64ed-4994-a101-addfaca5b0e4

## Build

See [releases](https://github.com/DrShahinstein/minesweeper-qt/releases/tag/Stable).

### Linux
```
$ mkdir build
$ cd build/
$ cmake ..
$ make
$ ./minesweeper

# or

$ ./run
```
